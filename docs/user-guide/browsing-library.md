# Browsing Your Library

Muzakily provides multiple ways to browse and discover music in your library.

## Songs View

The Songs view displays all songs in your library.

### Sorting

Click column headers or use the sort dropdown:

| Sort Option | Description |
|-------------|-------------|
| Title | Alphabetical by song title |
| Artist | Alphabetical by artist name |
| Album | Alphabetical by album name |
| Year | By release year |
| Date Added | Most recently added first |

### Filtering

Use the filter controls to narrow your view:

- **Search**: Type to filter by title, artist, or album
- **Genre**: Filter by music genre
- **Format**: Show only MP3, AAC, or FLAC files
- **Favorites**: Show only favorited songs

### Pagination

Large libraries are paginated. Use the controls at the bottom to:

- Navigate between pages
- Change items per page (25, 50, 100)

## Albums View

Browse your music organized by album.

### Album Grid

Albums display as a grid with cover art. Each card shows:

- Album cover
- Album name
- Artist name
- Release year

### Album Details

Click an album to see:

- Full-size cover art
- Complete track listing
- Total duration
- Play and shuffle buttons

### Playing an Album

- **Play All**: Click the play button to start from track 1
- **Shuffle**: Click shuffle to play in random order
- **Play from Track**: Click any track to start from that point

## Artists View

Browse your library organized by artist.

### Artist List

Artists appear alphabetically with:

- Artist image (if available)
- Number of albums
- Number of songs

### Artist Details

Click an artist to see:

- Artist biography (if enriched)
- Discography (albums by year)
- All songs

### Navigation

- Click an album to view its tracks
- Play all songs by the artist
- Add artist to favorites

## Smart Folders

Smart folders organize music based on file paths in storage.

### Folder Hierarchy

Folders appear in a tree structure matching your storage organization:

```
Music/
├── Rock/
│   ├── Classic Rock/
│   └── Alternative/
├── Pop/
└── Jazz/
```

### Browsing Folders

1. Click a folder to see its contents
2. View nested subfolders
3. See songs in the current folder
4. Song counts include subfolders

### Use Cases

- Organize by decade: `Music/1980s/`, `Music/1990s/`
- Organize by source: `Music/Vinyl Rips/`, `Music/CDs/`
- Organize by quality: `Music/Hi-Res/`, `Music/Standard/`

## Search

Quick search from anywhere in the app.

### Global Search

1. Press `/` or click the search icon
2. Type your query (min 2 characters)
3. Results appear instantly

### Search Results

Results are grouped by type:

- **Songs** - Matching song titles
- **Albums** - Matching album names
- **Artists** - Matching artist names

### Tips

- Search is typo-tolerant: "beetles" finds "Beatles"
- Partial matches work: "moon" finds "Dark Side of the Moon"
- Search across fields: "beatles abbey" finds songs by Beatles on Abbey Road

See [Search](search.md) for advanced search features.

## Context Menu

Right-click any item for options:

### Song Options

- Play
- Play Next
- Add to Queue
- Add to Playlist
- Add to Favorites
- Go to Artist
- Go to Album

### Album Options

- Play All
- Shuffle
- Add to Queue
- Add to Favorites
- Go to Artist

### Artist Options

- Play All Songs
- Shuffle All
- Add to Favorites

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `/` | Focus search |
| `Escape` | Close modal/clear search |
| `Enter` | Play selected item |
| `↑` `↓` | Navigate list |
| `Home` | Go to first item |
| `End` | Go to last item |
