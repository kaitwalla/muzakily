# Playlists

Create and manage playlists to organize your favorite music.

## Creating a Playlist

### From the Sidebar

1. Click **+ New Playlist** in the sidebar
2. Enter a name for your playlist
3. Optionally add a description
4. Click **Create**

### From a Song

1. Right-click any song
2. Select **Add to Playlist**
3. Choose an existing playlist or click **New Playlist**

## Managing Playlists

### Viewing a Playlist

Click a playlist in the sidebar to see:

- Playlist name and description
- Cover art (generated from songs)
- Total songs and duration
- Track listing

### Adding Songs

Several ways to add songs:

1. **Drag and Drop**: Drag songs onto the playlist in the sidebar
2. **Context Menu**: Right-click a song → Add to Playlist
3. **Multi-select**: Select multiple songs, right-click → Add to Playlist
4. **From Album**: Right-click an album → Add to Playlist

### Removing Songs

1. Open the playlist
2. Right-click a song
3. Select **Remove from Playlist**

Or hover over a song and click the X icon.

### Reordering Songs

1. Open the playlist
2. Drag songs to reorder
3. Drop in the desired position

Changes save automatically.

### Editing Playlist Details

1. Open the playlist
2. Click the edit icon (pencil) or right-click → Edit
3. Modify the name or description
4. Click **Save**

### Deleting a Playlist

1. Right-click the playlist in the sidebar
2. Select **Delete Playlist**
3. Confirm the deletion

**Note**: Deleting a playlist does not delete the songs.

## Playing Playlists

### Play All

Click the play button to start from the first song.

### Shuffle

Click the shuffle button to play in random order.

### Play from Song

Click any song in the playlist to start from that point.

### Add to Queue

Right-click the playlist → **Add to Queue** to append all songs to your current queue.

## Playlist Options

### Duplicate Playlist

1. Right-click the playlist
2. Select **Duplicate**
3. A copy is created with "(Copy)" appended

### Export Playlist

1. Right-click the playlist
2. Select **Export**
3. Choose format (M3U, CSV)

## Favorites Playlist

Your favorites are automatically collected:

- Songs, albums, and artists you've favorited
- Access via the Favorites section
- Updates automatically as you add/remove favorites

## Collaborative Features

### Sharing Playlists

If enabled by your administrator:

1. Open the playlist
2. Click **Share**
3. Choose visibility:
   - **Private**: Only you
   - **Shared**: Other users can view
4. Copy the share link

### Shared With Me

View playlists shared by other users in the **Shared** section.

## Tips

### Organizing Playlists

- Use descriptive names: "Workout Mix", "Road Trip 2024"
- Add descriptions for context
- Create themed playlists: by mood, activity, or genre

### Quick Add

While browsing, press `P` to add the selected song to your last-used playlist.

### Smart Playlists

For dynamic playlists that update automatically, see [Smart Playlists](smart-playlists.md).
